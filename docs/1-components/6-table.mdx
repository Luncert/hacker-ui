# Tables

Tables in Hacker UI are composable with 8 components:

- TableContainer
- Table
- TableRow
- TableHead
- TableHeaderCell
- TableBody
- TableBodyCell
- TableCaption

It has four variants:
- Outlined (default)
- Contained
- Striped
- Ghost

By default, Hacker UI tables are responsive (horizontally scrollable), with fixed first column on the left, and has hover effect on table body rows.
